## To-Do PWA

A Progressive Web App (PWA) for managing to-do lists, supporting offline access and installation on devices.

### Features
- Add, view, and manage to-do items
- User authentication (login/signup)
- Offline support via service worker
- Responsive design
- Installable on mobile and desktop

### Installation
1. Clone or download this repository.
2. Open the project folder in your code editor.
3. Serve the app using a local server (e.g., Live Server extension ).
4. Open `index.html` in your browser.

### Usage
- Sign up or log in to your account.
- Create new to-do items.
- View, edit, or delete existing items.
- Access your lists offline.
